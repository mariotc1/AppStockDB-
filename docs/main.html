main module

main.py

Este módulo gestiona la carga de sesión del usuario y el arranque de la ventana principal de la aplicación PyQt5.

Incluye funciones para: - Cargar el identificador de sesión guardado localmente. - Mostrar una pantalla de carga mientras se recuperan los datos del usuario desde la API. - Lanzar la ventana principal (MainWindow) con el usuario autenticado.

Requiere que el servidor Flask esté corriendo localmente (http://localhost:5000).
